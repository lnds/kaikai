# native-selfhost-baseline.txt — the expected-fail ratchet for the
# NATIVE SELF-HOST gate (issue #1021).
#
# The single non-comment, non-blank line below is an integer: the count
# of `unbound register` subset-gap aborts the in-process libLLVM native
# backend (`kaic2 --emit=native`) emits when it tries to compile the
# COMPILER ITSELF (`stage2/main.kai`, the modular entry point) — the one
# program no test exercised before this gate, so its KIR-node subset gap
# was silent. tools/test-native-selfhost-gate.sh measures and gates:
#
#   - count  > baseline  => FAIL (REGRESSION — native lost ground; a
#       change made the native backend reject a node it used to lower,
#       or grew the compiler's use of an already-unlowered node).
#   - count == baseline  => PASS (the documented expected-fail: the
#       known gap held steady; the self-compile still does not produce a
#       binary, but it got no worse).
#   - count  < baseline  => PASS, with a prompt to lower this file to the
#       new count (the ratchet tightens; part of the gap was closed —
#       mirror of tools/coverage-baseline.txt / native-parity-baseline.txt).
#   - count == 0         => PASS, "native self-host ACHIEVED" — the day
#       the gap closes and #1021 flips green-real. At that point the gate
#       should be upgraded to require the native object to LINK + RUN
#       (the issue's full milestone), and #1021 closed.
#
# This gate does NOT close the gap; it makes the gap VISIBLE and guards
# against regression. Closing the gap (driving this count to 0) lives in
# stage2/compiler/emit_native_fn.kai + kir_lower_*.kai and is separate,
# potentially large work tracked by #1021.
#
# Baseline measured on `main` (post-bump v0.97.0, commit 2497450d) with
# the KAI_LLVM=1 kaic2, deterministic across runs. Breakdown of the 25
# by register name (for context; the gate counts only the total):
#   n      x8   (let/var-bound value reads in arm bodies)
#   nm     x7
#   tyname x6
#   name   x2
#   eff    x2
# Identical to the 0.96-release measurement in #1021 — the gap has not
# moved between 0.96 and main.

25
